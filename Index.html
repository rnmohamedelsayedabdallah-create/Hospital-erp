🤖 Google Apps Script Generator — توليد الكود
انسخ هذا الكود وضعه في Google Apps Script ثم انشره كـ Web App للتكامل مع النظام. 
اختر نوع الكود لتوليده...
📌 خطوات النشر: 1) افتح Google Sheets → Extensions → Apps Script   2) الصق الكود → احفظ   3) Deploy → New Deployment → Web App → Execute as Me → Anyone   4) انسخ الـ URL وضعه في حقل "رابط Apps Script" أعلاه
🔗 Clone Code — كود المزامنة بين النسخ
توليد كود يمكن وضعه في نسخة أخرى من النظام لجلب البيانات من هذه النسخة أو إرسالها إليها. هذا يتيح تشغيل نسخ متعددة للوحدات المختلفة مع مزامنة مركزية. 
اضغط "توليد كود المزامنة" لبدء التوليد...
